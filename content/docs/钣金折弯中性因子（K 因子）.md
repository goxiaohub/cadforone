## 什么是 K 因子

钣金折弯时，材料内侧受压、外侧受拉，中间有一层**既不受压也不受拉的"中性层"**。中性层的位置用 **K 因子** 表示：

```
K = 中性层到内表面的距离 / 材料厚度
```

- **K = 0** → 中性层在内表面（材料被完全压缩）
- **K = 0.5** → 中性层在正中间（理想情况）
- **K 越大** → 中性层越靠近外表面，展开长度越长

## 使用方法

### 标准步骤

```
① 计算 R/T 比值（内弯曲半径 ÷ 材料厚度）
② 找到对应材料的 K 值（查综合表中的材料列或 K1 列）
③ K 值 × 材料厚度 → 得到中性层偏移距离
④ 在 CAD 中量出偏移后的折弯线长度 → 即为展开长度
```

### 折弯展开计算公式

```
L = L1 + L2 + (π × (R + K × T) × θ) / 180

其中：
  L  = 展开总长
  L1 = 直边段 1
  L2 = 直边段 2
  R  = 内弯曲半径
  K  = K 因子
  T  = 材料厚度
  θ  = 折弯角度（度）
```

**示例**（来自原表）：料厚 1mm，内 R 4mm，折弯角 120°，两直边各 30mm

```
展开长 = 30 + 30 + (3.14 × (4 + 0.33 × 1) × 120) / 180
       = 60 + (3.14 × 4.33 × 120) / 180
       = 60 + 9.07
       = 69.07 mm
```

## UG NX 中设置 K 因子

在 NX 钣金模块中：

```
文件 → 实用工具 → 钣金首选项（Sheet Metal Preferences）
  → 折弯系数（Bend Allowance）
    → 类型：折弯系数表（Bend Table）或公式（Formula）
    → 输入 K 因子值
```

也可以自定义折弯系数表（.bnd 文件），将上表中的数据导入，按材料自动匹配。

## K 因子变化规律总结

| R/T 范围 | K 值范围 | 说明 |
|:--------:|:--------:|------|
| < 0.3 | 0.23 ~ 0.30 | 小半径折弯，材料挤压严重 |
| 0.3 ~ 2.0 | 0.30 ~ 0.45 | 常规折弯范围，K 值随 R/T 增大而增大 |
| 2.0 ~ 5.0 | 0.45 ~ 0.48 | 大半径折弯，中性层接近中心 |
| > 5.0 | 0.48 ~ 0.50 | 半径很大，近似中性层在正中 |
